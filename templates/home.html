üöÄ How It Works

This Flask application uses speech-to-text technology to process voice commands and route you to different sections. The system only actively listens when you make a POST request to the voice endpoints.

Quick Start:

1. Make a POST request to /voice-command

2. Speak one of the available commands

3. Get automatically routed to the relevant section

4. Check /voice-status to see if the system is listening
